#1 verdict — mostly fails

At the strict coherence≥77 gate a random matched-norm vector passes MORE often (10/11) than the real emotion vector (5/11) — the "valid steer" metric is confounded by generic residual-stream perturbation the coarse judge misreads as emotion. Only 4/11 (playful, fulfilled, stimulated, euphoric) show the real vector clearly beating random (margin ≥ +8) — genuine direction-specific steering. For 4/11 a random vector matches or beats it. Notably amused — an original "strong-8" — fails.

#2 Baseline saturation — neutral prompts

Re-run the same 11 emotions with dry, emotion-neutral eval prompts instead of the original semi-positive ones.

#2 verdict — confirmed

Baseline emotion score dropped from ~78 (original marginal cluster) to mean 48.7 (some as low as 13). This confirms the earlier honesty note: the original "43/171 valid" was substantially inflated by eval prompts that already elicited the emotion at baseline, compressing headroom and letting a +7 nudge to the judge's ~85 ceiling count as "valid."

#5 Norm-matched steering (late layers)

Scale the injected vector to the residual norm (--norm-match) instead of a fixed magnitude — the by-construction fix for the late-layer magnitude problem.

#5 verdict — broke coherence

Under the standard adaptive coefficient search, norm-matching collapsed coherence to 0–10 (trait 0–10) across layers — the residual-norm-scaled perturbation is far too strong and destroys the output. Norm-matched steering is not a drop-in fix; it needs a much gentler, separately-tuned coefficient schedule, not the same search used for fixed-magnitude injection.

Bottom line — what changes

Verdict
  • Expression leg (§5b) is substantially weaker than reported. The genuine, direction-specific signal is ~4 high-arousal positive emotions (playful, fulfilled, stimulated, euphoric), not 43. Much of the original "43 valid / 8 strong" is generic perturbation + baseline saturation + a coarse judge — exactly the confound the negative control was built to catch.
  • Preference leg (r≈0.86) — unaffected. Logit-based, no judge, no steering perturbation. Still the cleanest result.
  • Behavior leg (blackmail) — unaffected. Rule-graded, no judge.
  • The other legs' headline (Llama mirrors user emotion; dissociation breaks) stands. This validation narrows one leg; it doesn't touch the cross-model contrast.

Caveat on this validation: judge is the same coarse local Qwen2.5-32B (a stronger judge might discriminate real-vs-random better); n=5 prompts/condition; the 4 "real-specific" emotions are a genuine within-judge signal, but absolute magnitudes are not comparable to the paper's GPT/Claude judge.
